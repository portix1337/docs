Nydo Theme Documentation

A comprehensive guide to installing and customizing the Paymenter Nydo theme. This documentation covers everything from initial setup to advanced component customization.

Installation Instructions

Step 1: Download the Template

Get the template files (HTML, CSS, assets).

Step 2: Locate Paymenterâ€™s Themes Directory

Navigate to your Paymenter installation:

cd /var/www/paymenter/themes
Step 3: Upload Files

Upload the Nydo folder into the Paymenter theme folder.

Note: You can use sftp like WinSCP for uploading.

Step 4: Build the Theme

Run the following commands to build the theme:

curl -fsSL https://deb.nodesource.com/setup_22.x | sudo bash -
sudo apt install -y nodejs
cd /var/www/paymenter
npm install
npm run build nydo
Step 5: Enable the Theme

In the Settings Page in your admin side, change the theme from default to nydo.

Troubleshooting

  • Make sure you are using Node.js v20 or above.
  • Check your terminal for missing dependencies or errors.
  • Refer to the Nydo support channels if you encounter issues.

About the Nydo Theme

The Nydo theme is a modern, minimalist dark theme for Paymenter, crafted with a focus on delivering a clean, fast, and highly intuitive user experience. We believe that a great interface should be elegant and effortless, allowing users to focus on what matters most without unnecessary distractions.

Our Philosophy

Cleanliness & Simplicity

Every element is thoughtfully designed to be clean and straightforward, reducing visual clutter and enhancing readability.

Performance

We've optimized every line of code to ensure the theme is lightweight and loads incredibly fast, providing a snappy experience for all users.

User-Centric Design

The theme is built with the user in mind, featuring an intuitive layout and easy navigation to streamline common tasks.
